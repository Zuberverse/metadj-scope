# MetaDJ Scope

**Last Modified**: 2025-12-26 17:00 EST

Hackathon exploration project for the Daydream Scope Track (Interactive AI Video Program). Building a Scope-generated MetaDJ avatar from webcam input for a real-time streaming demo.

## Status
- **Phase**: Local development environment ready
- **Stack**: Next.js 15 + TypeScript + Tailwind CSS
- **Direction**: MetaDJ avatar generated by Scope from live webcam input
- **RunPod**: ✅ Deployed (`metadj-scope` on RTX 5090)
- **Reference Image**: MetaDJ v7.0 avatar for VACE identity consistency

## Hackathon Context
- **Program**: Daydream 2025 Interactive AI Video Program (Scope Track)
- **Timeline**: Dec 22 - Jan 8 (two-week sprint)
- **Prizes**: $2,500 / $1,750 / $750 for top 3
- **Details**: See `docs/scope.md`

This project connects to the broader MetaDJ ecosystem:

## Quick Start

### RunPod Deployment (Recommended for hackathon)

**Prerequisites:**
- RunPod account with credits
- HuggingFace account with read token

**Steps:**
1. Open [RunPod Scope Template](https://runpod.io/console/deploy?template=daydream-scope)
2. Create HuggingFace token at [huggingface.co/settings/tokens](https://huggingface.co/settings/tokens) with **read** permissions
3. Select GPU (≥24GB VRAM, recommend RTX 4090)
4. Click "Edit Template" → Add `HF_TOKEN` env var with your HuggingFace token → Save
5. Click "Deploy On-Demand"
6. Access at `https://your-instance-id.runpod.io:8000`

**Full setup details**: See `docs/research.md` (RunPod Setup section)

### Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Type check
npm run type-check
```

Then open http://localhost:3000 to see the Avatar Studio.
